Telegram Message Scheduler Bot

Your Personal Message Reminder

Schedule messages to be sent back to you at the perfect time.

Features

  • Schedule messages to be sent back to you
  • Forward existing messages to schedule them
  • Flexible time specification (5m, 2h, 1d, etc.)
  • List all your scheduled messages

How to Use

  1. Start a conversation with the bot by searching for it on Telegram
  2. Send /start to begin and see available commands
  3. Forward any message or send a new one to the bot
  4. Specify when you want to receive the message (e.g., "2h" for 2 hours)
  5. The bot will confirm your scheduling and deliver the message at the specified time

Commands

Command Description
/start Start the bot and see the welcome message
/help Display help information and usage instructions
/schedule Start the process of scheduling a new message
/list Show all your currently scheduled messages
/cancel Cancel the current scheduling operation
Time format examples: 5m (5 minutes), 2h (2 hours), 1d (1 day), 30s (30 seconds), 3h 30m (3 hours and 30 minutes)
